💻 10 Fun Coding Facts

Fact 1
The first computer bug was a real moth found in a Harvard Mark II computer in 1947.
Fact 2
The “Hello, World!” program became popular from Brian Kernighan’s book in 1972.
Fact 3
The first programmer was Ada Lovelace in the 1800s, working on Charles Babbage’s Analytical Engine.
Fact 4
The Python language was named after “Monty Python,” not the snake.
Fact 5
The first website ever created is still online: info.cern.ch.
Fact 6
There are over 700 programming languages in existence today.
Fact 7
The term “open source” was coined in 1998 to promote free software collaboration.
Fact 8
The first video game, “Spacewar!”, was created in 1962 by MIT students.
Fact 9
GitHub hosts over 200 million repositories, making it the largest code-sharing platform.
Fact 10
The longest piece of code ever written is the Google codebase, with billions of lines.
